# 🌿 花園兒童遊戲 — 使用說明

三個可直接用瀏覽器開啟的兒童遊戲網頁（無需伺服器或網路）。

---

## 1. 🌱 花園種花 — `plant.html`

**目標**：在 9 個花盆中都種出花朵。

**玩法**：
1. 點擊 **空的土盆** → 種下一顆種子 🌰
2. 再點擊種子 → **澆水** 💧，植物會成長：
   - 🌰 種子 → 🌱 嫩芽 → 🌿 花苞 → 🌸 開花
3. 每盆需要**澆水 3 次**才會開花
4. 9 盆全部開花就贏了 🎉

**統計**：🌸 已開花數量、💧 總澆水次數

---

## 2. 🐞 花園捉蟲 — `bugcatch.html`

**目標**：30 秒內點擊盡量多的小蟲。

**玩法**：
1. 遊戲開始後，蟲會在花園各處出現並**四處移動**
2. 用滑鼠/手指 **點擊蟲** 來捉住牠們
3. 蟲會在一段時間後**自動消失**，要趕快點！
4. 同時間場上最多 5 隻蟲

**蟲蟲分數**（由高到低）：

| 蟲 | 分數 |
|---|---|
| 🦋 蝴蝶 | 15 |
| 🐝 蜜蜂 | 12 |
| 🐞 瓢蟲 | 10 |
| 🐜 螞蟻 | 8 |
| 🐛 毛毛蟲 | 8 |
| 🐌 蝸牛 | 5 |

**計時條**會顯示剩餘時間，結束後會根據總分給予評語。

---

## 3. 🥕 蔬菜記憶 — `veggie.html`

**目標**：翻開所有蔬菜配對。

**玩法**：
1. 共有 16 張牌（8 種蔬菜，各 2 張），**面朝下**排列
2. 每次**翻開 2 張**
3. 如果**相同** → 配對成功，卡片保持翻開 🎯
4. 如果**不同** → 牌會晃動後蓋回去 🔄
5. 全部 8 組配對完成就贏了 🎉

**統計**：❓ 步數（翻牌次數）、✅ 配對進度

---

## 技術細節

- 純 HTML + CSS + JavaScript，無任何外部依賴
- 所有遊戲皆為**繁體中文**
- 支援**手機觸控**與**電腦滑鼠**
- 每個遊戲都有「重新開始」按鈕
- 可在任何現代瀏覽器中執行
