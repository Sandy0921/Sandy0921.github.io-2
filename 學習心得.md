## 一、初學網頁的挑戰

還記得第一次打開 VS Code，面對漆黑的編輯器和一個閃爍的游標，內心充滿了茫然。那些由英文字母和符號組成的程式碼就像外星語言一樣難以理解。為了搞懂 HTML 標籤為什麼要成雙成對、路徑怎麼設定圖片才不會破圖、CSS 的選擇器到底在選什麼，花了不少時間摸索。甚至常因為漏掉一個 `>` 或打錯一個單字，整個網頁就亂掉，卡關許久。

## 二、從零到有的成就感

但當我第一次成功寫出 `<h1>Hello World</h1>`，在瀏覽器中看到那行字真的跳出來時，那種震撼與驚喜難以言喻——就像手握魔法棒，在原本空無一物的數位世界裡創造出東西來。

隨著學習的深入，我開始能夠寫出更複雜的頁面結構、用 CSS 把畫面變得漂亮、用 JavaScript 加入互動功能。從最基本的靜態網頁，到後來能夠做出有動畫、有分數計算、有遊戲規則的互動網頁，每一次的進步都帶來了極大的滿足感。

## 三、花園遊戲的開發歷程

### 🌱 花園種花

這是我設計的第一個遊戲。核心機制很簡單——點擊種植、澆水、開花。但在實作過程中，我學到了如何用 JavaScript 管理不同階段（種子 → 嫩芽 → 花苞 → 開花）、如何用 CSS transition 做出生長的動畫效果、以及如何在對的時機觸發勝利條件。

### 🐞 花園捉蟲

限時遊戲的挑戰在於時間邏輯與動態生成。我學會了用 `setInterval` 和 `setTimeout` 控制蟲的生成與消失、追蹤遊戲狀態（進行中 / 結束）、即時更新分數與時間條。為不同蟲設定不同分數也讓遊戲更有策略性。

### 🥕 蔬菜記憶

經典翻牌遊戲讓我學到了如何管理陣列與隨機洗牌（Fisher-Yates 演算法）、控制遊戲流程鎖定（避免玩家在比對期間繼續翻牌）、以及用 CSS 動畫給予明確的視覺回饋（翻轉、彈跳、晃動）。

## 四、技術成長

透過這次專題，我學到了：

1. **HTML 語意結構** — 建立乾淨、可讀的頁面架構
2. **CSS 排版與動畫** — Flexbox、Grid、`@keyframes`、transition
3. **JavaScript 邏輯控制** — 陣列操作、事件處理、計時器、狀態管理
4. **響應式設計** — 用 `@media` 讓手機與電腦都有好的遊玩體驗
5. **除錯能力** — 遇到 Bug 時如何一步步找出問題、修正錯誤

## 五、心得總結

這次製作三個花園兒童遊戲，讓我深刻體會到：**寫網頁不只是敲打鍵盤，而是一場把抽象思考變成具體作品的創作旅程。**

從一開始面對空白畫面的茫然困惑、到後來看著三個完整的遊戲在瀏覽器上順利運作，這趟從零到有的過程帶給了我滿滿的成就感。雖然過程中遇到了大大小小的 Bug——動畫不對、計時器打架、配對邏輯出錯——但每一次解決問題的瞬間，那種喜悅與滿足都讓我更有動力繼續學下去。

未來我想加入更多功能（音效、排行榜、難度選擇），讓這套遊戲變得更完整。
