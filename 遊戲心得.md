# 💭 花園兒童遊戲 — 製作心得

## 設計理念

三個遊戲圍繞「花園」主題，各自訓練不同的能力：

| 遊戲 | 核心能力 | 設計靈感 |
|---|---|---|
| 🌱 花園種花 | 耐心、因果觀察 | 讓孩子體驗「種植→照顧→開花」的歷程 |
| 🐞 花園捉蟲 | 反應力、專注力 | 限時 + 亂動目標，增加挑戰與樂趣 |
| 🥕 蔬菜記憶 | 記憶力、配對邏輯 | 經典翻牌遊戲，以蔬菜取代一般符號更貼近主題 |

## 技術亮點

- **零依賴**：純 HTML/CSS/JavaScript，開箱即用
- **響應式設計**：手機觸控與電腦滑鼠皆可遊玩
- **動畫回饋**：每項操作都有即時視覺回饋（彈跳、晃動、紙屑、澆水動畫）
- **分數差異化**（捉蟲遊戲）：不同蟲給不同分數，鼓勵孩子認識與區分物種

## 改進方向

- 可加入音效（Web Audio API 產生簡單音效不需外部檔案）
- 種花遊戲可加入更多植物品種與成長階段
- 可加入排行榜（localStorage 儲存最佳成績）
- 可加入難度選擇（更多卡片、更短時間、更多花盆）

## 開發觀察

用 emoji 作為遊戲角色是一個很棒的選擇——不需要繪圖資產、跨平台顯示、孩子一看就懂。三款遊戲雖然玩法不同，但共用一致的視覺風格（圓角、柔和漸層、彩色陰影），讓整套遊戲有一致感。
